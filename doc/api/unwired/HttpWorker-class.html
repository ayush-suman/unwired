HttpWorker

HttpWorker class Null safety

HttpWorker is the base class for all the workers that process the requests. This can be extended to create your own implementation for processing requests.

For example, you can create a HttpWorker that processes requests on a pool of Isolates with an implementation of load balancer to balance the request across Isolates.
